Diagnostic de connexion Google Sheets
1 Prérequis : Sheet partagé en lecture

Le Sheet doit être partagé avec « Tout le monde possédant le lien → Lecteur ».

Dans le Sheet : bouton Partager → Accès général → Tout le monde possédant le lien → rôle Lecteur → OK
⚠️ Si le Sheet est « Limité », la connexion échouera même si vous êtes connecté à Google.
2 Lancer le diagnostic complet

Le test inspecte la réponse brute de Google Sheets et vous explique précisément ce qui fonctionne ou bloque.
